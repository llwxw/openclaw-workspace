# Graph View

> 原文提到: "checking the graph view"

## 作用

可视化 wiki 页面之间的链接关系，帮助发现：
- 孤立的页面
- 核心枢纽页面
- 连接紧密的页面群

## Obsidian 实现

```
打开方式: Ctrl+G (或 Cmd+G)
```

## 分析要点

- 中心节点: 被最多页面链接
- 边缘节点: 孤立页面
- 簇: 紧密相关的页面群
