# Book Notes

> 原文: "Reading a book: filing each chapter as you go"

## 笔记结构

```
books/
  {book-name}/
    summary.md      # 书籍摘要
    chapter-1.md    # 第一章笔记
    chapter-2.md    # 第二章笔记
    characters.md   # 人物
    themes.md       # 主题
    timeline.md     # 时间线
    connections.md  # 与其他书的联系
```

## 模板

```markdown
# {章节名}

## 关键概念
-

## 重要引用
> 

## 思考
-

## 与其他章节的联系
-
```

## 完成后

- 创建书籍维基
- 连接人物、主题
- 与其他书籍建立关系
