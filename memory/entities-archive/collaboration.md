# Collaboration

> 团队协作机制

## 原文对应

> "Business/team: an internal wiki maintained by LLMs"

## 协作模式

### 人类角色

- 添加 source
- 提问和探索
- 审核 AI 修改

### AI 角色

- 维护 wiki
- 处理更新
- 解决冲突

## 协作流程

```
人类添加 Source → AI Ingest → 更新 Wiki → 人类审核
                    ↓
人类提问 → AI Query → 生成答案 → 人类反馈
```

## 权限管理

- 读取: 团队成员
- 写入: AI (受监督)
- 审核: 人类
