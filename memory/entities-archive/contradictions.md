# ⚔️ Contradictions

> 矛盾与Claim 管理

## 矛盾类型

| 类型 | 描述 | 处理方式 |
|------|------|----------|
| **事实冲突** | A 说 X，B 说非 X | 保留双方，标注日期和来源 |
| **过时信息** | 新数据 vs 旧结论 | 标记过时，保留参考 |
| **观点差异** | 不同角度的分析 | 并存，注明立场 |

## 标记格式

```markdown
### 矛盾记录

| 日期 | 来源A | 来源B | 状态 |
|------|-------|-------|------|
| 2026-04-11 | 旧假设 | 新发现 | 待确认 |

> [假设前提] 旧假设基于 X 条件...
```

## Claim 强度

| 标记 | 含义 |
|------|------|
| 🟢 确认 | 有明确来源支持 |
| 🟡 推测 | 有限证据 |
| 🔴 存疑 | 与其他信息矛盾 |
