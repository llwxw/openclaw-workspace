# 🧘 Wiki Mindset

> 维护心态与哲学

## 核心理念

| 心态 | 说明 |
|------|------|
| **长期主义** | wiki 价值随时间增长，不求短期回报 |
| **渐进构建** | 从一页开始，慢慢丰富 |
| **不完美 OK** | 可以后面再改进，不要追求一步完美 |
| **LLM 为我服务** | 我是 Wiki 的维护者，不是被 Wiki 绑架 |

---

## Do's

| ✅ 应该 | 原因 |
|---------|------|
| 先行动，再优化 | 完美主义是敌人 |
| 写比不写好 | 粗糙的笔记比没有好 |
| 链接比不链接好 | 少点也可以，先建立连接 |
| 定期回顾 | 看到自己的积累会有动力 |

---

## Don'ts

| ❌ 不应该 | 原因 |
|----------|------|
| 追求一次完美 | wiki 是迭代的 |
| 什么都想记 | 只记值得的 |
| 过于复杂 | 简单有效最好 |
| 忽略维护 | 只建不用会荒废 |

---

## 心理阶段

| 阶段 | 心态 |
|------|------|
| 新鲜期 | 认真建，热情高 |
| 沉默期 | 没什么感觉，正常用 |
| 价值期 | 开始感受到积累的好处 |
| 依赖期 | 离不开了 |
