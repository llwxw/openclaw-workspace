# Web Clipper

> 原文提到: "Web Clipper for capturing content from the web"

## 工具选择

| 工具 | 说明 |
|------|------|
| Obsidian Web Clipper | 浏览器插件，直接捕获到 Obsidian |
| Notion Web Clipper | 类似功能 |
| 手动复制 | 最简单可靠 |

## 捕获流程

1. 安装 Web Clipper 扩展
2. 选中网页内容
3. 发送到 wiki 对应文件夹
4. LLM 读取并处理
