# Scenarios

> LLM Wiki 的应用场景

根据 LLM Wiki 原文，有 5 个主要应用场景：

## 1. Personal (个人知识)

构建个人知识管理系统：
- 目标管理
- 健康追踪
- 心理分析
- 自我提升

### 本项目实现

- [[user.md]] - 用户
- [[self-improving]] - 自我改进
- [[preferences]] - 偏好

## 2. Research (研究)

深入研究某个主题：
- 阅读论文
- 分析报告
- 构建thesis

### 本项目实现

- [[ai-research]] - AI 研究
- [[case-study]] - 研究案例

## 3. Reading a Book (读书笔记)

阅读时构建维基：
- 章节记录
- 人物分析
- 主题总结

### 扩展

可以创建 book-wiki.md

## 4. Business/Team (团队知识库)

团队共享知识：
- Slack 集成
- 会议记录
- 项目文档

### 本项目实现

- [[feishu]] - 飞书集成
-  - GitHub

## 5. Competitive Analysis (竞争分析)

商业分析场景：
- 市场调研
- 对比分析

## 本项目对应的场景

主要实现 **Personal** 和 **Research** 场景。
