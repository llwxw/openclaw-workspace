# 🔄 Auto Learning

> 自动化学习循环

## 自动化触发

| 触发条件 | 动作 |
|----------|------|
| 新会话启动 | 检查是否有待处理内容 |
| 对话中获取新知识 | 自动判断是否写 wiki |
| 定期 (cron) | 检查环境 + lint |
| 用户明确说"记住" | 强制 ingest |

---

## 自动学习来源

| 来源 | 监控方式 |
|------|----------|
| 用户对话 | 每次响应时检查是否有新知识 |
| 工作区新文件 | 定期扫描 workspace/ |
| skills/ 更新 | 检查新技能文档 |
| docs/ 新内容 | 检查新增文档 |

---

## 自动执行流程

```
1. 启动时
   → 扫描工作区变化
   → 检查是否有新知识

2. 对话中
   → 识别有价值的信息
   → 自动写入 wiki (不询问)

3. 定期
   → 运行 Lint 检查
   → 更新 log
```

---

## 融合优先机制

根据 **LLM Wiki 原文**：「You never (or rarely) write the wiki yourself」

### 添加规则

| 情况 | 处理 |
|------|------|
| 可融合到现有页 | ✅ 合并到现有页面 |
| 可升级现有页 | ✅ 更新替代 |
| 无法融合 | ❌ 不添加、不记录 |
