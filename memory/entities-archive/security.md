# Security

> Wiki 安全机制

## 安全原则

1. **备份** - 定期备份
2. **验证** - 修改前验证
3. **回滚** - 保留历史版本
4. **隔离** - 测试/生产分离

## 权限控制

- 读取: 开放
- 写入: 仅 AI/自动化脚本
- 删除: 仅手动确认
