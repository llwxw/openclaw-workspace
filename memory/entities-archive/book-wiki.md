# 📖 Book Wiki

> 读书知识库

## 适用场景

- 读一本书时，边读边记
- 按章节记录要点
- 构建角色、主题、情节的关联
- 读完后拥有完整的伴读 wiki

## 类似案例

[Tolkien Gateway](https://tolkiengateway.net) — 粉丝维护的托尔金百科
- 数千个互关联的页面
- 角色、地点、事件、语言

## 页面结构

```
book-书名/
├── index.md         # 书籍概览
├── chapter-1.md     # 第 N 章笔记
├── chapter-2.md
├── characters.md    # 角色列表
├── places.md        # 地点
├── themes.md        # 主题分析
├── plot.md          # 情节线
└── timeline.md      # 时间线
```

## 记录原则

| 内容 | 记录方式 |
|------|----------|
| 章节要点 | 每章一文件 |
| 新角色 | 创建独立页面 |
| 伏笔 | 链接到后续章节 |
| 主题 | 独立概念页 |

## 与普通 wiki 的区别

| 维度 | 普通 Wiki | Book Wiki |
|------|-----------|-----------|
| 时间 | 持续 | 读书周期 |
| 结构 | 按概念 | 按章节+角色 |
| 完整性 | 逐步完善 | 读完后完整 |
