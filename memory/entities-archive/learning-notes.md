## 页面结构


## 笔记格式

### 康奈尔笔记法

```markdown
# 主题

## 线索
[关键词/问题]

## 笔记
[主要内容和解释]

## 总结
[简短摘要]
```

### 详细笔记

```markdown
# [课程/书籍名称]

## 核心概念
1. 概念 A
2. 概念 B

## 详细解释
...

## 思考与问题
...

## 相关页面
- [[related]]
```
