# Advanced Search

> 高级搜索功能

## 搜索类型

### 1. 精确搜索
搜索确切关键词。

### 2. 模糊搜索
语义相似匹配。

### 3. 跨页面搜索
全文检索。

## 高级语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `""` | 精确匹配 | `"task orchestrator"` |
| `tag:` | 按标签 | `tag:framework` |
| `link:` | 按链接 | `link:openclaw` |
| `-` | 排除 | `-deprecated` |

## 搜索结果排序

- 相关度
- 更新时间
- 访问频率
