# 🔄 Knowledge Reuse

> 知识的复用与重组

## 复用方式

| 方式 | 说明 | 示例 |
|------|------|------|
| **直接引用** | 链接到已有页面 | 读取 [[entity]] |
| **综合** | 合并多个来源 | 创建新页面 |
| **对比** | 比较异同 | 创建对比表 |
| **派生** | 从 A 推到 B | 推理新结论 |

---

## 复用流程

```
1. 遇到问题/主题 X
      ↓
2. 搜索 wiki 中相关页面
      ↓
3. 读取 2-3 个相关页
      ↓
4. 综合形成答案
      ↓
5. 判断是否写回:
   - 是 → 创建/更新页面
   - 否 → 直接回答
```

---

## 复用的价值

- **时间**: 不需要每次重新理解
- **深度**: 跨领域连接产生新洞察
- **质量**: 综合多个可靠来源

---

## 复用案例

```
用户: "什么是 LLM Wiki?"
→ 读取 llm-wiki.md (基础)
→ 读取 ingest.md (工作流)
→ 读取 second-brain-report.md (实践)
→ 综合成一个完整的回答
→ 如果回答得好 → 写回 wiki
```
