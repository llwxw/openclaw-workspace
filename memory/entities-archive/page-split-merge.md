# 🔀 Page Split & Merge

> 页面拆分与合并

## 何时拆分

| 信号 | 操作 |
|------|------|
| 一个页面超过 500 行 | 拆分成多个 |
| 内容涉及多个主题 | 按主题拆分 |
| 页面太宽泛 | 拆成专业页面 |

## 何时合并

| 信号 | 操作 |
|------|------|
| 多个页面内容高度重复 | 合并 |
| 相关页面但访问量都低 | 合并 |
| 过细的分类 | 合并 |

## 拆分步骤

```
1. 创建新页面 (按模板)
2. 复制相关章节到新页
3. 原页保留精简版 + 链接到新页
4. 检查所有引用旧页的链接
5. 更新 MEMORY.md
```

## 合并步骤

```
1. 选择主页面
2. 从其他页面复制内容
3. 删除被合并的页面
4. 更新所有指向被删除页面的链接
5. 更新 MEMORY.md
```
