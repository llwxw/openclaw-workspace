# ✍️ Writing Wiki

> 写作与创作知识库

## 适用场景

- 写博客/文章
- 创作小说
- 记录灵感
- 整理素材

---

## 页面结构

```
writing/
├── ideas/            # 灵感库
├── drafts/           # 草稿
├── published/        # 已发布
├──素材/             # 写作素材
└── research/         # 主题研究
```

---

## 内容类型

| 类型 | 说明 |
|------|------|
| 灵感 | 片段想法 |
| 大纲 | 文章结构 |
| 草稿 | 完整文章 |
| 已发布 | 带有链接 |

---

## 灵感记录格式

```markdown
# 灵感：[标题]

## 核心想法
一句话概括

## 扩展
[详细展开]

## 标签
#灵感 #待写
```
