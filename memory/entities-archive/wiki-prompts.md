# 📝 Wiki Prompts

> 用户引导 Wiki 的常用指令

## 常用指令

| 场景 | 指令 | 效果 |
|------|------|------|
| 记住这个 | "记住..." 或 "记录..." | Ingest 到 wiki |
| 查 wiki | "wiki 里关于 X" | 搜索 + 回答 |
| 写回来 | "把这个写回 wiki" | 创建/更新页面 |
| 检查健康 | "wiki lint" | 运行检查 |
| 快速了解 | "快速了解 X" | 用 quick-start |

---

## 完整指令集

### 知识摄取

```
"记住我今天学到了 X"
"把这个文章存到 wiki"
"记录这个偏好：..."
```

### 知识查询

```
"wiki 里有没有关于 X 的内容？"
"给我总结一下 wiki 里关于 X 的"
"X 和 Y 有什么区别？wiki 知道吗？"
```

### 知识写回

```
"这个答案很棒，写回 wiki"
"创建个页面记录这个"
```

### 维护

```
"运行 wiki lint"
"检查死链"
"看看 wiki 最近有什么更新"
```

---

## 自然语言触发

不需要学命令，自然语言也可以：

| 自然语言 | 意图 | 执行 |
|----------|------|------|
| "这个很重要千万别忘了" | 记住 | Ingest |
| "帮我查一下之前有没有记录过 X" | 查询 | Search |
| "你觉得这个答案值得保存吗" | 判断写回 | 询问 |
| "我的 wiki 现在啥情况" | 状态 | Lint |
