# 📋 Quality Audit

> 质量审核标准

## 审核维度

| 维度 | 权重 | 检查项 |
|------|------|--------|
| 完整性 | 30% | 是否涵盖核心内容 |
| 准确性 | 30% | 信息是否正确 |
| 可读性 | 20% | 格式是否清晰 |
| 链接 | 20% | 链接是否有效 |

---

## 评分标准

| 分数 | 等级 | 说明 |
|------|------|------|
| 90-100 | A | 优秀，直接可用 |
| 70-89 | B | 良好，稍作修改 |
| 50-69 | C | 一般，需要补充 |
| <50 | D | 需要重构 |

---

## 审核流程

```
1. 随机抽取 10% 页面
2. 按维度评分
3. 汇总统计
4. 制定改进计划
5. 执行优化
```

---

## 工具辅助

```bash
# 完整性检查
wc -l *.md | awk '$1<50 {print}'

# 链接检查
./scripts/check-dead-links.sh
```
