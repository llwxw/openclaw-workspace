# 📦 Product Wiki

> 产品管理知识库

## 适用场景

- 记录产品需求
- 管理产品路线图
- 收集用户反馈
- 整理竞品分析

---

## 页面结构

```
product/
├── roadmap.md        # 产品路线图
├── requirements/     # 需求池
├── users/            # 用户反馈
├── competitors/      # 竞品分析
└── metrics/          # 关键指标
```

---

## 内容类型

| 类型 | 内容 |
|------|------|
| PRD | 需求文档 |
| 竞品 | 对比分析表 |
| 用户反馈 | 原始记录 + 分析 |
| 指标 | DAU/MAU/转化率 |

---

```markdown
# [产品功能名称]

为什么做这个功能

## 目标

[详细描述]

## 指标
| 指标 | 目标 | 实际 |
|------|------|------|
```
