# 💬 Feedback Mechanism

> 反馈机制

## 反馈类型

| 类型 | 来源 | 处理 |
|------|------|------|
| 用户纠正 | 用户说我错了 | 写入 corrections.md |
| 用户补充 | 用户补充信息 | 更新页面 |
| 用户建议 | 用户建议改进 | 评估后执行 |
| 系统问题 | Lint 报错 | 立即修复 |

---

## 反馈流程

### 用户纠正

```
用户: "你说得不对"
    ↓
我: 承认错误
    ↓
写入 corrections.md
    ↓
更新相关页面 (如需要)
    ↓
感谢用户
```

### 用户建议

```
用户: "建议添加 X"
    ↓
评估: X 是否有价值
    ↓
有价值 → 创建/更新
    ↓
回复用户进展
```

```markdown
## [2026-04-11] 反馈

```
