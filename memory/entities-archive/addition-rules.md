# 添加规则

> 融合优先，不超过 30 个核心页面

## 核心原则

1. **融合优先** - 新内容优先融合到现有页面
2. **30 个上限** - 核心页面不超过 30 个
3. **归档机制** - 不常用的移至归档
4. **绝不偷懒** - 严格执行，不走过场

## 添加流程

```
发现新内容
    ↓
判断：是否可融合到现有页面？
    ↓ 是 → 融合，结束
    ↓ 否
判断：核心页面是否 < 30？
    ↓ 是 → 添加为新页面
    ↓ 否 → 拒绝/归档
```

## 禁止添加

- 已存在的内容不添加
- 可融合的不添加
- 超过30个后不添加

## 归档恢复

需要时从 entities-archive/ 恢复。
