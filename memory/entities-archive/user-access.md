# 👁️ User Access

> 用户如何查看 Wiki

## 当前方式

| 方式 | 说明 |
|------|------|
| 直接读文件 | 用户可随时读取 workspace 下的 .md 文件 |
| 问我 | "wiki 里关于 X 的内容是什么？" |
| 搜索 | 用户也可以用 openclaw memory search |

---

## 推荐工作流

### 双屏模式（理想）

```
┌──────────────────┐   ┌──────────────────┐
│   我（对话）      │   │   Obsidian       │
│                  │   │   (实时浏览)     │
│ - 回答问题       │ ← │   - 看修改结果   │
│ - 写入 wiki     │   │   - 跟随链接     │
│                  │   │   - Graph View  │
└──────────────────┘   └──────────────────┘
```

---

## 用户可做的事情

| 操作 | 方法 |
|------|------|
| 查看 wiki 内容 | 直接打开 .md 文件 |
| 搜索 | `openclaw memory search "关键词"` |
| 提问 | 问我wiki相关内容 |
| 手动编辑 | 用任意编辑器 |
| 导出 | 复制 .md 内容 |

---

## 信息同步

我修改 wiki 后，用户**立即可见**，因为都是本地文件。

---

## 隐私说明

- Wiki 在本地 (`~/.openclaw/workspace/memory/`)
- 不上传云端
- 用户拥有完全控制
