### 学习点
1. 安装技能时优先选择用户目录安装（~/.local/），避免权限问题
2. coding-agent 需要明确指定 workdir 参数，否则代理不知道在哪个项目工作
3. Claude Code 使用 `--permission-mode bypassPermissions --print` 参数，无需 PTY
4. 用户对文件安全非常重视，所有操作都要透明且可逆
