# ⚠️ Corrections

> 错误与纠正记录

## 概述

用户的每一次纠正都是学习机会。

| 规则 | 说明 |
|------|------|
| 每次纠正 | 立即写入 ~/self-improving/corrections.md |
| 分析根因 | 为什么错？ |
| 避免重复 | 更新认知 |

---

## 格式

```markdown
## [2026-04-11] 纠正内容

### 错误
...我之前说/做了什么...

### 正确
...应该是...

### 根因
- 假设了错误的前提
- 遗漏了关键信息

### 改进
- 以后遇到 X 场景要 Y
```

## 历史记录

已整合到 `~/self-improving/corrections.md`

## 常见错误类型

| 类型 | 示例 |
|------|------|
| 臆造事实 | 说错了 API 版本 |
| 忽略前提 | 没看清用户条件 |
| 过度假设 | 自动脑补不存在的内容 |
| 记忆偏差 | 记错了之前对话内容 |

---

## 避免策略

1. 不确定 → 说"不确定"
2. 引用 → 给出验证方法
3. 假设 → 标注 `[假设前提]`
4. 记住 → 立即写入文件
