# Heartbeat Configuration

## 心跳规则

### 任务进度上报格式
```
[PROGRESS] <percent>% | <step>/<total> | <description>
```

### 心跳间隔
- 默认: 30秒
- 最小: 10秒
- 最大: 300秒

### 超时处理
| 情况 | 处理 |
|------|------|
| 30秒无输出 | 发送"仍在运行"提示 |
| 60秒无产出 | 警告用户 |
| 超时前5秒 | 发送即将超时警告 |

### 环境变量
- `SUBAGENT_MEMORY_LIMIT_MB`: 子代理内存限制 (默认 2048)
- `PROGRESS_MIN_INTERVAL_MS`: 进度上报最小间隔 (默认 1000)

### 注意事项
- 长时间任务需定期输出进度
- 关键里程碑应立即上报
- 错误信息立即上报，不等待心跳
